# Recursive Echo
かつて私にとってプログラミングとは、指先を通じて自らの思考をダイレクトに組み上げていく、職人的な手仕事でした。

しかしAIが瞬時にコードを生成する現在、その風景は一変しました。 私が入力するのは、「プログラムの構造」から「AIへの指示」へと変わり、AIに委ねて待つ時間が生まれました。プログラムとの直接的な繋がりが途絶えたような感覚が残ります。

私は今、どのように「もの」を作っているのでしょうか。本作『Recursive Echo』はプログラムを用いて作業ログを可視化し、制作プロセスを省みる実験です。

図像は、「可視化プログラム」そのものがどのように組み上げられてきたのか、その過程を示しています。このプログラムの制作では私はAIに指示を出し、あるいは直接コードを記述します。それらの行為は同時にログとして記録され、まさに制作中の同プログラムにより即時に可視化されます。私はその図像を見つめ、また次の指示や修正を加える。こうした重層的なループのなかでの試行の軌跡が、一枚の図像として定着しています。

そしてこれは、プログラマーとしての次のcraftsmanshipを、私自身が現在進行形で探り続けている最中の記録でもあるのです。

- **【作品の読み方（凡例）】**

    本作は、AIとの往復のなかで生まれた「コードの生成・修正・削除」の記録を、以下の規則で幾何学模様へと変換しています。

    - **各グリッドと配置：** 実制作時間を均等に分割したもの（左から右、上から下へと時間が進行）
    - **グリッドの太枠：** AIに対してプロンプトを入力したタイミング
    - **斜線：** その時間帯に変化したプログラムの量
    - **点と線：** その時間帯に追加されたプログラムの量
    - **空白：** その時間帯に削除・破棄されたプログラムの量
